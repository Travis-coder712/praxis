# Praxis

Practical AI, learned by doing.

**Live at:** https://travis-coder712.github.io/praxis/

## What this is

A hands-on AI literacy program for working professionals. 10 modules, 75+ lessons across three tiers
(Beginner / Intermediate / Advanced), with experimentation tasks woven into every lesson.

Module 1 ("How AI actually works") is fully built in v0.1. Modules 2–10 are outlined in the curriculum
and will ship progressively.

## Stack

React 19 · TypeScript · Vite 6 · Tailwind 4 · vite-plugin-pwa

## Local development

```bash
npm install
npm run dev
```

Deploys automatically to GitHub Pages on every push to `main`.
